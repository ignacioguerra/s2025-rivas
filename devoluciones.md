### Generales

- Las imágenes se pueden otimizar un poco más, audio también.
- Se intenta usar la tipografía Merriweather, pero no está cargada.
- Da la sensación de que falta la página individual de la obra (parece que son cliqueables, pero no)

### CSS

- Bien en general. Utiliza REMs , animaciones y varios de los elementos trabajados en clase.
- Muchos estilos se repiten en los diferentes archivos. Eso es un problema a la hora de mantener el codigo, porque un simple cambio podría impactar en muchos archivos. Es recomendable en casos así tener un CSS para estilos compartidos (header, estructura, etc.) y se pueden mantener los otros CSS para las particularidades de cada página

### Responsive

- Bien. Aplica estilos fluidos y media-queries
- Adapta la experiencia interactiva acorde a cada dispositivo

### Uso de Astro o JS

- Slideshow con reproductor de audio. El botón mute es algo confuso.
